# Online Exam & Assessment Platform

## English | [中文](#中文)

An online exam and assessment platform built with Go, Gin, PostgreSQL, and Redis.

### Features
- Exam creation and management (multiple question types)
- Question bank with categories and tags
- Auto-grading and manual scoring
- Anti-cheating (timer, tab-switch detection, randomized questions)
- Real-time exam monitoring
- Score analytics and ranking
- Certificate generation
- Support for timed and practice exams

### Quick Start
```bash
docker-compose up -d
go run cmd/api/main.go
```
